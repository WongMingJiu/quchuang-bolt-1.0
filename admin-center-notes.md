# 管理中心当前版本说明文档

## 一、版本定位

当前版本的“管理中心”属于：

- **第一版前端可见 / 可交互骨架**
- 并且已经开始具备部分后端与数据库基础

也就是说，它不是一个完全静态的设计图页，而是已经具备：

- 页面入口
- 页面骨架
- 交互抽屉
- Tabs 切换
- 时间筛选
- 数据库 migration
- 后端接口骨架

但它还不是最终的“完整运营后台”。

## 二、当前页面结构

管理中心当前是一个单页结构，内部通过 `Tabs` 切换三大功能模块：

### 1. 团队仪表盘
### 2. 个人仪表盘
### 3. 模型权限与额度

页面位置：
- 左侧导航新增：
  - `管理中心`

### 已经具备

## 七、当前推荐使用方式

### 当前适合做的事情
- 看页面结构
- 验证交互路径
- 演示信息架构
- 作为下一阶段真实数据接入的 UI 基线

### 当前不建议认为已经完成的部分
- 真实统计结果
- 真实权限保存
- 真实个人 / 团队数据口径

## 八、当前版本价值

这一版管理中心的价值在于：

1. 把原本零散的后台需求聚合到一个统一入口
2. 把三大后台能力的页面结构先固化
3. 把交互路径先打通：
   - 查看详情
   - 编辑
   - 筛选
4. 为后续真实数据接入准备好了足够稳定的 UI 容器和接口骨架
